ICT- INFORMATION COMMUNICATION TECHNOLOGY

It deals with the use of different communication technologies such as mobile phones, telephone, internet, etc. to locate, save, send, and edit information.

Information: process data.

Communication: exchanging of information or news.

Technology: application of scientific knowledge for practical purposes, especially in industry

Internet: is the global system of interconnected computer networks that uses tcp/ip to link billions of devices worldwide.


ICT in the Philippines- Department of Information Communications Technology (DICT) is responsible for formulation, recommendation and implementation of policy that will promote development and global competitiveness of the country’s ICT.

ICT in the Philippines – called as “ICT HUB OF ASIA”

World Wide Web - An information system on the Internet which allows documents to be connected to other documents by hypertext links, enabling the user to search for information by moving from one document to another.

Webpage - a hypertext document connected to the World Wide Web.

Website - a location connected to the Internet that maintains one or more pages on the World Wide Web.


Three (3) kinds of website:

1. Static web – web 1.0 - A static website contains Web pages with fixed content. Each page is coded in HTML and displays the same information to every visitor. Static sites are the most basic type of website and are the easiest to create.

2. Dynamic web - web 2.0- Dynamic websites contain Web pages that are generated in real-time. These pages include Web scripting code, such as PHP or ASP. When a dynamic page is accessed, the code within the page is parsed on the Web server and the resulting HTML is sent to the client's Web browser.

Features:

Folksonomy: it allows you to categorize and arrange information.

Rich user experience: responsive to user’s input.

User Participation: owner of the website is allowed to put content.

Long Tail: service that are offered on demand rather than on a one-time purchase.

Software or Service: subscribed to software when only needed rather than purchasing.

Mass participation: diverse information sharing through universal web access.


3. Semantic web – web 3.0- a proposed development of the World Wide Web in which data in web pages is structured and tagged in such a way that it can be read directly by computers.

Problem’s with Web 3.0

-Compatibility
-Security
-Vastness
-Vagueness
-Logic
Trends in ICT

Convergence - the process or state of converging.

Social Media- websites and applications that enable users to create and share content or to participate in social networking.

Six types of social media:

-Social networks
-Bookmarking sites
-Social News
-Media Sharing
-Micro blogging
-Blogs and Forums

Mobile Technologies-technology used for cellular communication. Many experts believe that the future of computer technology rests in mobile computing with wireless networking. Mobile computing by way of tablet computers is becoming more popular.

Assistive Media-Assistive technology is any device, software, or equipment that helps people work around their challenges. Some examples of assistive technology are text-to-speech and word prediction. Assistive technology includes low-tech tools, too, like pencil grips.
